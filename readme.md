
# Web socket.io

a websocket application for handle of real-time communication with prototype application.

---

### Application Requirements

-  **Node.js** version >= 12.22.6
-  **npm** version >= 6.14


### Application Setup

> make sure already clone the project directory.

```bash
npm install
```

### Local run server
> local port is `3001`

```bash
npm run dev
```
  

### Deployment Setup

```bash
npm install
```
```bash
npm run start
```
